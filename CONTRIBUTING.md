## Local build
1. Setup local database:  
In `dev-mssql/` folder run
```bash
docker-compose build
docker-compose up
```
2. Start backend:  
In `backend/` folder run
```bash
npm install
npm run dev
```
3. Start frontend:  
In `frontend/` folder run
```bash
npm install
npm start
```
